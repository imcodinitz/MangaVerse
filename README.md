# MangaVerse

A MangaDex-inspired web project.

As for now, we are pretty heavily AI-based due to shortage amount of personels (of only 3)
Our current AI materials all came from ChatGPT, Deepseek and with the addition of Cursor AI to help with live-testing, coding (especially with API integration) and debugging.


As of the first version's release of 1.0:
- Achieved the basic front layout, making it eye-catchy to users.
- Featuring carousels of currently popular, recently uploaded mangas that users can pick from. (and a little all-time manga rankings according to views/likes)
- Implementing Kitsu API.
- Simply Designed Favicon.

As for the second version's release of "Reworked MangaVerse", with the help of AI tools, we have achieved more since the first one, which includes:
- Added a warm welcome message in the front page.
- Reworking on the carousels' main functions: Infinite auto-scroll feature to make the website feels more lively + hovering animations - user-interactive.
- Fixing the overall look of the navigation bar (a styled search bar + Home, Genres buttons and a button which links to Kitsu's API documentation)
- Added a "default user" button, follow with a login/register forms. (haven't added)
- Added a few glossy and hovering effects to make the website feel much more polished and smooth, while keeping everything clean.
